Instructions

  1. Enter all information as shown in examples, on booking page.
  2. If you want an indoor lane, prefix the lane number with a 9.
  3. If you want to start at a half hour (ex 9:30) do not type in 9:30. Type in 9 and toggle the add half hour button.
  4. Once all information is typed in correctely, hit submit and wait for 4:45 AM the following day.
